## MongoDB 
- is distributed document database
- NoSQL 
- uses json-like documents with optional schemas

## MongoDB Structure
- cluster
    - database
        - collection (SQL: table)
            - documents (SQL: records)

## Model
- model - schema
- Mongoose generates collections in mongodb
    - _id - unique id for each document
